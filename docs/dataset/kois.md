# Kepler Objects of Interest (KOIs)

KOIs are well vetted, periodic, transit-like events in the Kepler data. The Kepler Project identifies these objects from the TCE list for further vetting. Some objects will be flagged as false positives.

All KOI tables are displayed in a single, interactive environment with separate tabs for each table (e.g., Q1-6, Q1-8, cumulative, etc). Column selection and filtering applies to all tabs, allowing easier comparison of values across KOI tables.

## MCMC Posterior Chains

The following files contain Markov Chain Monte Carlo (MCMC) posterior chains from fits to the KOIs. Compressed tar files are available for direct download, by wget script for the complete set (single script), or by wget script with one tar file per KOI (per object script).

| Release | Description | Compressed File (.tar) | Download Script (wget) | Documentation |
| --- | --- | --- | --- | --- |
| Q1-Q16 | This file contains the MCMC posterior chains for all Q1-Q16 KOIs using fits to the Data Release 20-23 light curves and the Q1-Q16 Stellar Properties. | [download](https://exoplanetarchive.ipac.caltech.edu/data/KeplerData/General/mcmcChains.tgz)<br>(~8 GB) | [single script](https://exoplanetarchive.ipac.caltech.edu/data/KeplerData/General/KOI_MCMC_wget.bat)<br>[per object script](https://exoplanetarchive.ipac.caltech.edu/data/KeplerData/General/KOI_MCMC_dr22_distinct_wget.bat) | [KSCI-19084](https://exoplanetarchive.ipac.caltech.edu/docs/KSCI-19084-002.pdf) |
| Q1-Q17 DR 24 | This file contains the MCMC posterior chains for the new Q1-Q17 DR 24 KOIs using fits to the Data Release 24 light curves and the Q1-Q17 Stellar Properties. | [download](https://exoplanetarchive.ipac.caltech.edu/data/KeplerData/General/q1q17dr24_mcmcChains.tgz)<br>(~8 GB) | [single script](https://exoplanetarchive.ipac.caltech.edu/data/KeplerData/General/KOI_q1q17dr24_MCMC_wget.bat)<br>[per object script](https://exoplanetarchive.ipac.caltech.edu/data/KeplerData/General/KOI_MCMC_dr24_distinct_wget.bat) | [KSCI-19084](https://exoplanetarchive.ipac.caltech.edu/docs/KSCI-19084-002.pdf) |
| Q1-Q17 DR 25 | This file contains the MCMC posterior chains for the new Q1-Q17 DR 25 KOIs using fits to the Data Release 25 light curves and the Q1-Q17 Stellar Properties. | [download](https://exoplanetarchive.ipac.caltech.edu/data/KeplerData/General/q1q17dr25_mcmcChains.tgz)<br>(~35 GB) | [single script](https://exoplanetarchive.ipac.caltech.edu/data/KeplerData/General/KOI_q1q17dr25_MCMC_wget.bat)<br>[per object script](https://exoplanetarchive.ipac.caltech.edu/data/KeplerData/General/KOI_MCMC_dr25_distinct_wget.bat) | [KSCI-19113](https://exoplanetarchive.ipac.caltech.edu/docs/KSCI-19113-001.pdf) |
