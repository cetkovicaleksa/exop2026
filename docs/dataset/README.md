This directory contains reference documentation for the Kepler Objects of Interest (KOI) dataset used in this project.

The files in this directory were converted<sup>[w2md]</sup> from documentation published by NASA's Exoplanet Archive and are provided for convenience and offline access.

**Contents:**
* [purpose.md](purpose.md) — Overview of the dataset and its intended use.
* [columns.md](columns.md) — Description of dataset columns and their meanings.
* [kois.md](kois.md) — Information about Kepler Objects of Interest (KOIs) and related terminology.

**Note:**  
These documents are vendor-provided reference materials and are not authored or maintained as part of this project. Refer to the original NASA Exoplanet Archive for the most up-to-date documentation.
